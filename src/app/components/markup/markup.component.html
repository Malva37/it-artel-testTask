    4. за умови такого коду: 
          
<div class="container">
  <div class="block"></div>
  <div class="block"></div>
  <div class="block"></div>
</div>
    • варіант 1(inline-block) 
                
.block {
  display: inline-block;
  width: 20px;
  height: 20px;
  border: 1px solid black;
  box-sizing: border-box;
  }
.container {
  width: 60px;
  font-size: 0;
}
    • варіант 2(flex) 
                
.container {
  display: flex;
  justify-content: space-between;
  width: 60px;
  height: 20px;
  border: 1px solid black;
  align-items: center;
  box-sizing: border-box;
}
.block {
  width: 20px;
  height: 20px;
  border: 1px solid black;
  box-sizing: border-box;
}
    • варіант 3(grid) 
                  
.container {
  display: grid;
  grid-template-columns: repeat(3, 20px);
  grid-gap: 0;
  width: 60px;
}
.block {
  width: 20px;
  height: 20px;
  border: 1px solid black;
}
    • варіант 4(float)(не часто використовується. Опціонально можна додати в розмітку ще одиг блок, після трьох блоків з класом .clear, який має властивість clear: both, щоб зупинити float) 
                  
.container {
  width: 60px;
  height: 20px;
  float: left;
}
.block {
  float: left;
  width: 20px;
  height: 20px;
  border: 1px solid black;
  box-sizing: border-box;
